Open Log File¶

To open a log file and add it to the current session, select Session | Open, then choose the correct schema from the drop down & any preferred log builder and click the Open … button. A standard File Open dialog will appear, allowing one or more log files to be opened.

A log builder consists of a log file pattern theme and one or more view pattern themes. When the log file is opened, the log file pattern is applied, and one view is opened for each listed view pattern. This permits one or more views to be opened with predefined markers, searches etc. The available log builders are defined by the schema.

Navigation Themes¶

Colours and styles in NLV can be stored as a theme. This includes:

  • Markers

  • Trackers, and

  • Searching & Hiliting

The gallery of available themes can be found at the Session | Themes panel. Themes can be activated, copied, renamed and deleted. Either right click the theme in the list and choose the appropriate option, or enter the equivalent keyboard shortcut.

For information on editing navigation themes, see the Log File and Views pages.
